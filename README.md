## Project Structure

- A folder by subject.  
* By subject, there are one or more examples of C programming, Arduino programming, and image files with schematics.  
* Filename convention:  
    `subject-<example-number>.<extention>`  
    * The extensions are:
        * .c for C programming;
        * .ino for Arduino programming;
        * .png/.jpg/.svg/.pdf for schematics.
